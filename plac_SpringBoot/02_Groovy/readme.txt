★ Spring Boot とは？
 - もっと効率的に必要最小限の作業でSpringMVCを使ったWebアプリケー ションを構築できるようにしよう
 - アノテーションによる開発 
   - ｢XMLによる設定ファイルだらけ｣ → ｢アノテー ションによる、設定ファイルを使わない｣

★ SpringBootアプリケーションについて
 - Groovyによるアプリケーション 
 - Javaによるアプリケーション 

★ GroovyアプリとJavaアプリ
 - Groovy
   -「Javaで開発するSpringBootアプリの基本部分を簡単 に作れるようにした、簡易版SpringBootアプリ｣
   - GroovyアプリでSpringBootを使った開発の基本 → ｢Javaを利用した、本格的な開発｣への予備知識が一通り身につく


★ SpringBootCLIの用意
 - ダウンロード先
    - http://docs.spring.io/spring-boot/docs/current/reference/html/getting-started-installing-spring-boot.html#getting-started-installing-the-cli
 - spring-boot-cli-1.5.1.RELEASE-bin.zip　をダウンロードし、展開して配置のみ
   → G:\tools\dev\spring-1.4.0.RELEASE

★ Groovyスクリプトを作成する
 → ※ ./Groovy01 に作成
 
★ app.groovyを実行する
 $ G:\tools\dev\spring-1.4.0.RELEASE\bin\spring run app.groovy
 → http://localhost:8080/にアクセスで「Hello!」
